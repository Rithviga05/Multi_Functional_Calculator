markdown# Multi-Functional Python Calculator

This is a learning project I built to practice Python basics, handle inputs, and explore the built-in `math` module. 

### What I Learned:
* How to use `import math` to unlock advanced functions like logarithms and trigonometry.
* How to use conditional `if` statements to prevent program crashes (like dividing by zero).
* Troubleshooting indentation errors and syntax mistakes.

*Note: This program was developed as a learning exercise with structural guidance and code debugging assistance from an AI collaborator.*
